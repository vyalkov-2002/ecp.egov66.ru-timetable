Расписание колледжей и техникумов Свердловской области
======================================================

``ecp.egov66.ru-timetable`` — это Python-библиотека для получения расписания
студента и преподавателя из личного кабинета `ГИС СО "ЕЦП"`_, написанная путём
нехитрого реверс-инжиниринга :)

.. _`ГИС СО "ЕЦП"`: https://lk.ecp.egov66.ru/

С помощью этой библиотеки вы можете создавать собственные сервисы для просмотра
расписания. Ознакомиться с примерами можно на странице :doc:`showcase`.

Начните изучение с раздела :doc:`getting-started` и продолжите разделом
:doc:`usage`.

По всем вопросам можно писать на почту vyalkov.2002@mail.ru или в Telegram
`@sysrqtech <https://t.me/sysrqtech>`_.
