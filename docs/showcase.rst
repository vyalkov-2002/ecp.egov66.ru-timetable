Примеры сервисов
================

Вот несколько связанных между собой проектов, центральным компонентом которых
является данная библиотека.

Скриншоты скрыты под спойлеры. Нажмите на них, чтобы раскрыть.


Расписание66
------------

* Исходный код: https://altlinux.space/acme-corp/egov66-timetable-chatbot

Чат-бот для просмотра расписания в популярных мессенджерах. Хотя в боте можно
подписаться на оповещения об изменениях в расписании, сам бот эту рассылку не
осуществляет, этим должен заниматься другой компонент.

Бот использует расписание и список групп из базы данных, так что требуется
компонент, который будет регулярно обновлять расписание.

ИАТ-Расписание
--------------

* Исходный код: https://altlinux.space/acme-corp/iat-timetable

Скрипт для периодического обновления данных. Получает все расписание, создает
страницу с выбором группы или преподавателя и публикует сайт на Codeberg Pages.
Кроме того, с помощью :doc:`коллбэков <usage>` скрипт вносит расписание в базу
данных и отправляет в мессенджеры уведомления об изменениях в расписании.

     :alt: Скриншот страницы с расписанием группы на неделю.

ИАТ-Табло
---------

* Исходный код: https://altlinux.space/acme-corp/iat-tablo

Скрипт для генерации PDF-файла с расписанием для всех групп. Использует
расписание и список групп из базы данных и библиотеку `FPDF2`_ для создания PDF.

.. _`FPDF2`: https://py-pdf.github.io/fpdf2/index.html

.. rst-class:: details

Скриншоты
  .. image:: img/iat-tablo.png
     :alt: Страница из файла. Расписание представляет собой таблицу, в которой
           столбцы - это номера групп. Каждый такой столбец делится еще на два
           столбца - номер аудитории и название предмета. Строки - это пары по
           порядку. Отдельными строками прописаны названия дней недели. Внизу
           дисклеймер.
